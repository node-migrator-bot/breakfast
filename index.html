breakfast by rasantiago

breakfast = CoffeeScript, HAML and a side of SASS (and other scripting languages) in one delicious node.js compiler ....... maybe I should call it breakfast burrito script

With a plethora of great scripting languages to create Javascript, HTML and CSS it seemed like it would be nice to have an environment where I could leverage all them. The effort here is simple: leverage script compilers (e.g. coffeescript, haml, etc.) and node.js to create a dev util for righting awesome code at the speed of thought....or at least the speed of coffee!!!

Dependencies

node.js for sure and the available compilers for coffeescript, HAML and SASS. Soon breakfast will be setup as a node package so you can have NPM take care of installing all the dependencies. The only other dependency are the right TextMate Bundles for the unified code dev environment.

Install

Will soon be packaged so you can have NPM install it for you.
